# 开始使用

推荐使用Docker容器部署本项目.

## 1.安装Docker及git

官网下载对应版本的安装包即可,Linux系统的不同发行版安装方法略有不同.

## 2.克隆项目到本地

打开命令行输入如下命令:
`git clone https://github.com/wx-shen/bootstrapslice`

## 3.由源码构建容器,启动镜像

对于Linux操作系统,在命令行键入`cd bootstrapslice-master && bash ./build.sh`
对于Windows操作系统,双击build.bat文件

## 4.访问网页

浏览器输入"localhost:8081",即可访问
